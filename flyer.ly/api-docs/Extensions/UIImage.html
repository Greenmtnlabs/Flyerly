  • sh_convertFramesToGif(_:settingsManager:cancelHelper:resultHandler:)

    Converts the given frames (images) to GIF format.

    Error codes description.

    - 800 - Empty gif frames array.
- 810 - Cannot create CGImageDestination.
- 811 - The image doesn't have a valid CGImage.
- 820 - Request was Cancelled.
- 821 - Cannot retrieve the image from the storage.
- 830 - Failed to Finalize the Image Destination.

    Declaration

    Swift

    class func sh_convertFramesToGif(gifFrames: [GFGifFrame], settingsManager: GFAnimationSettingsManager, cancelHelper: GFCancelOperationsHelper, resultHandler: ((data: NSData?, error: NSError?) -> Void)) -> Void

    Parameters

    gifFrames

    The gif frames which are used to retrieve images required for GIF creation.
    settingsManager

    Settings manager object that is used to retrieve all the required properties of the animation frames.
    cancelHelper

    The object that is used to provide the current operation identifier to the calling function to make it possible cancel the task on demand.
    resultHandler

    A closure that is called when the animation is calculated. It provides error object to check whether the operation was completed successfully. Data object contains the calculated animation. It is called on main thread.
