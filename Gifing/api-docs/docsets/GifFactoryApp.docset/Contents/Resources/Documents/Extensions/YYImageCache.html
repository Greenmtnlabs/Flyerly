YYImageCache

class YYImageCache : NSObject
  • sh_setImage(_:forGifFrame:createThumbnailInMemoryWithSize:resultHandler:)

    Asynchronously stores the image to the cache for the given key.

    resultHandler parameters:

    • gifFrame: The GIF frame that was requested to be stored in the cache.
    • error: The NSError object containing the error information.

    Error codes description.

    • 400 - Failed to Retrieve GIF Frame Cache Keys.
    • 401 - Failed to Resize Thumbnail.
    • 402 - Failed to Unwrap Image Data.

    Declaration

    Swift

    func sh_setImage(image: UIImage, forGifFrame gifFrame: GFGifFrame, createThumbnailInMemoryWithSize size: CGSize?, resultHandler:((gifFrame: GFGifFrame, error: NSError?) -> Void)?)

    Parameters

    image

    The image that must be stored into the cache.
    gifFrame

    The object that describes the GIF properties.
    size

    The size in points of the thumbnail that must be stored in the memory cache.If nil than the fullsized image will be stored in memory instead. If size is not specified the image won’t be stored in memory.
    resultHandler

    The closure that is called when the image has been loaded from the cache.
